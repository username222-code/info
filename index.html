Deploy free on Cloudflare

This project is set up as a static site, which Cloudflare Pages supports directly.[web:113]

  1. Upload these files to a GitHub repository.[web:118]
  2. In Cloudflare, open Workers & Pages and connect that repository.[web:118]
  3. Use no framework preset, set build command to exit 0, and output directory to . for a plain static HTML site.[web:113]
  4. Deploy and Cloudflare will give you a free pages.dev URL, with free custom domains also supported on the free plan.[web:113][web:115][web:122]
